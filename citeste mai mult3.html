Eliminarea fracției: În primul rând, vom scăpa de fracție prin înmulțirea ambelor părți ale inegalității cu x + 1, dar trebuie să avem grijă la semnul acestuia. Dacă x + 1 > 0, vom înmulți fără a schimba semnul inegalității, iar dacă x + 1 < 0, semnul inegalității se va schimba. Vom analiza ambele cazuri:

Cazul 1: x + 1 > 0 (adică x > -1)
În acest caz, putem înmulți ambele părți ale inegalității cu x + 1:

3 < 4(x + 1)
Distribuirea: Aplicăm distribuirea pe partea dreaptă:
3 < 4x + 4
Izolarea lui x: Aducem termenii cu x pe o parte și termenii numerici pe cealaltă parte:
3 - 4 < 4x
-1 < 4x
Rezolvarea pentru x: Împărțim ambele părți la 4:
-1⁄4 < x

Cazul 2: x + 1 < 0 (adică x < -1)
În acest caz, vom înmulți ambele părți ale inegalității cu x + 1, dar semnul inegalității se va schimba:

3 > 4(x + 1)
Distribuirea:
3 > 4x + 4
Izolarea lui x:
3 - 4 > 4x
-1 > 4x
Rezolvarea pentru x:
x < -1⁄4

Concluzie: Soluția generală a inegalității este:
x > -1⁄4, dar trebuie să ținem cont de restricția x ≠ -1, deoarece în cazul în care x = -1, x + 1 devine 0 și fracția nu ar fi definită.

Astfel, soluția finală este:
x > -1⁄4, x ≠ -1.
