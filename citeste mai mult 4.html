Vom rezolva această inegalitate pas cu pas:

Găsirea valorilor critice: Primul pas este să identificăm valorile pentru care numărătorul și numitorul sunt zero. Aceste valori sunt punctele critice ale inegalității.

Numărătorul: 𝑥 + 2 = 0 ⇒ 𝑥 = −2
Numitorul: 𝑥 − 1 = 0 ⇒ 𝑥 = 1
Aceste valori sunt importante, deoarece ele pot schimba semnul fracției.

Analiza semnelor: Vom împărți linia numerelor în intervale pe baza acestor valori critice și vom analiza semnul expresiei în fiecare interval. Cele trei intervale sunt:
(−∞,−2), (−2,1), (1,∞).
Trebuie să verificăm semnul fracției în fiecare dintre aceste intervale, ținând cont de semnul numărătorului și al numitorului.

Intervalul (−∞,−2):
Alege un punct din acest interval, de exemplu 𝑥 = −3.
Numărătorul: (−3 + 2) = −1 (negativ)
Numitorul: (−3 − 1) = −4 (negativ)
Fracția −1/−4 = 1/4 este pozitivă. Deci, în intervalul (−∞,−2), fracția este pozitivă.

Intervalul (−2,1):
Alege un punct din acest interval, de exemplu 𝑥 = 0.
Numărătorul: (0 + 2) = 2 (positiv)
Numitorul: (0 − 1) = −1 (negativ)
Fracția 2/−1 = −2 este negativă. Deci, în intervalul (−2,1), fracția este negativă.

Intervalul (1,∞):
Alege un punct din acest interval, de exemplu 𝑥 = 2.
Numărătorul: (2 + 2) = 4 (positiv)
Numitorul: (2 − 1) = 1 (positiv)
Fracția 4/1 = 4 este pozitivă. Deci, în intervalul (1,∞), fracția este pozitivă.

Valorile critice:
La 𝑥 = −2, fracția devine 0/−3 = 0, care este valid, deci 𝑥 = −2 este inclus în soluție.
La 𝑥 = 1, fracția nu este definită, deoarece împărțirea la 0 nu este permisă, deci 𝑥 = 1 nu este inclus în soluție.

Concluzie:
Soluția inegalității 𝑥 + 2/𝑥 − 1 ≥ 0 este:
𝑥 ∈ (−∞,−2]∪(1,∞).
